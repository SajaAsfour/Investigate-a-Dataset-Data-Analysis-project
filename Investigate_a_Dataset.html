Introduction¶

Dataset Description¶

The dataset used in this project is the TMDb Movies Dataset, which contains information about approximately 10,000 movies collected from The Movie Database (TMDb). The dataset provides detailed information about movies including their financial performance, popularity, ratings, production details, and release information.

Each row in the dataset represents a single movie, while each column represents a specific attribute related to that movie.

Columns Description¶

id: Unique identifier for each movie

imdb_id: Unique movie identifier from IMDb

popularity: Popularity score based on user engagement and views

budget: Estimated production budget of the movie

revenue: Total revenue generated by the movie

original_title: Original title of the movie

cast: List of actors involved in the movie

homepage: Official website of the movie

director: Director of the movie

tagline: Promotional tagline of the movie

keywords: Keywords describing the movie content

overview: Brief summary of the movie plot

runtime: Duration of the movie in minutes

genres: Movie genres separated by a pipe (|) symbol

production_companies: Production companies involved in the movie

release_date: The date the movie was released

vote_count: Number of user votes received

vote_average: Average user rating score

release_year: Year the movie was released

budget_adj: Budget adjusted for inflation (in 2010 dollars)

revenue_adj: Revenue adjusted for inflation (in 2010 dollars)

Question(s) for Analysis¶

Main Question:¶

  • What factors are associated with higher movie revenue?

Supported Questions:¶

  1. Is there a relationship between movie budget and revenue?
  2. Do certain movie genres generate higher revenue than others?
  3. Does movie popularity influence revenue?
  4. Is there a relationship between audience ratings and movie revenue?

In this investigation, revenue will be treated as the dependent variable, while budget, genres, popularity, and vote_average will be treated as independent variables.

Identified Data Issues Before Cleaning¶

After looking at the data, several issues with the quality of the data were identified:

  • There are missing values found in various columns, such as homepage, tagline, keywords, production_companies, cast, director, and genres.

  • The duplicated row found within the data set.

  • There are some numeric values such as budget, revenue, and runtime, which have zeros; these zeros are not realistic and can affect the results of the analysis. These problems will be addressed in the subsequent process of data cleaning.

Data Cleaning¶

Removing Duplicate Rows¶

The dataset was tested against duplicate rows to ensure data quality. Generally, a duplicate record in a dataset distorts statistical analysis and yields a wrong result outcome; so, for this reason, the dataset was worked on to exclude duplicate rows.

In [8]:
df = df.drop_duplicates()

Duplicate rows were successfully removed from the dataset. This ensures that each movie is represented only once in the analysis.

Fixing Data Types¶

Some columns required conversion to appropriate data types to allow accurate analysis. The release_date column was converted to datetime format, while release_year and runtime were converted to numeric values.

In [9]:
df['release_date'] = pd.to_datetime(df['release_date'], errors='coerce')
df['release_year'] = pd.to_numeric(df['release_year'], errors='coerce')
df['runtime'] = pd.to_numeric(df['runtime'], errors='coerce')

Data types were successfully converted, which will allow proper time-based and numerical analysis.

Handling Invalid Zero Values¶

Columns such as budget, revenue, and runtime contained zero values, which are unrealistic for movies. These values were replaced with missing values (NaN) to prevent misleading analysis results.

In [10]:
zero_columns = ['budget', 'revenue', 'budget_adj', 'revenue_adj', 'runtime']

for col in zero_columns:
    df[col] = df[col].replace(0, np.nan)

Zero values were replaced with missing values to improve data accuracy and reliability.

Removing Rows with Missing Key Values¶

Rows with missing values in critical columns such as revenue, budget, popularity, vote_average, and genres were removed. These variables are essential for answering the research questions.

In [11]:
df = df.dropna(subset=['revenue_adj', 'budget_adj', 'popularity', 'vote_average', 'genres'])

Rows with missing key values were removed to ensure the dataset is suitable for analysis.

Creating Primary Genre Column¶

Since the genres column contains multiple values separated by a pipe symbol (|), a new column called primary_genre was created to simplify analysis by selecting the first listed genre.

In [12]:
df['primary_genre'] = df['genres'].str.split('|').str[0]

A new column was created to simplify grouping movies based on their main genre.

Conclusions¶

This analysis considered various factors that could potentially affect the revenue generated from movies, using the TMDb Movies dataset.

These results show that the production budget has a moderately positive association with the revenues, since the correlation is approximately 0.57. This means that movies with larger financial investments are apt to generate higher revenues.

Movie popularity was also moderately positively related to revenue, with a correlation of approximately 0.55. A movie that receives more audience attention and interest is very likely to result in better box office performance.

The genre analysis, meanwhile, showed the highest returns for Adventure, Animation, and Family movies. It could also be interpreted that such genres cater to the mass population and yield better economic returns.

In contrast, audience ratings were only weakly, albeit positively, correlated with revenue (correlation ≈ 0.27). While it is possible that more highly rated films have increased revenue, such factors cannot be used to strongly predict success.

Overall, the results suggest that the role of financial investment and audience popularity is more significant than that of audience ratings.

However, it is also important to note that this is exploratory analysis based on correlations. Therefore, we cannot say for sure that higher budgets or popularity actually lead to higher revenue. There could be other determinants of success, such as marketing, competition, distribution, international market, etc., which contribute to the overall business.

Limitations¶

There are, however, a few limitations with this analysis. One of these limitations is that many rows were deleted during cleaning, which decreased the overall number of rows. Again, there are limitations with utilizing less data, which can create an inability to generalize. A second limitation involves the fact that there were no marketing expenses, audience demographics, or international performance provided by the dataset. A third limitation involves the fact that no statistical tests were conducted.

Further research could take into account additional variables, possibly using more advanced statistical models to ascertain what makes movies successful.
