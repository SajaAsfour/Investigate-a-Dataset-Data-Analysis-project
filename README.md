# 📊 Investigate a Dataset – Data Analysis Project

## 📌 Project Overview

This project performs an **exploratory data analysis (EDA)** using Python and Jupyter Notebook.
The goal is to investigate multiple real-world datasets, clean the data, analyze relationships between variables, and draw meaningful insights using statistical methods and visualizations.

The analysis was implemented using **Python, Pandas, NumPy, and Matplotlib** inside a Jupyter Notebook environment.

## 📊 Datasets Used

### 1️⃣ NCIS and US Census Data

* **gun_data.csv**
* **US_Census_Data.csv**

Used to explore relationships between gun purchase background checks and demographic statistics in the United States.

---

### 2️⃣ Medical Appointment Dataset

* **noshowappointments-kagglev2-may-2016.csv**

Contains information about medical appointments and whether patients showed up or not.

Used to investigate questions such as:

* What factors influence appointment attendance?
* Does age or medical condition affect no-shows?

---

### 3️⃣ TMDb Movie Dataset

* **tmdb-movies.csv**

Contains movie metadata including:

* Budget
* Revenue
* Popularity
* Ratings
* Release year

Used to explore trends in movie performance and popularity.

---

## 🔍 Analysis Process

The analysis follows these steps:

1️⃣ **Data Loading**
Import datasets using Pandas.

2️⃣ **Data Cleaning**

* Handle missing values
* Remove duplicates
* Convert data types
* Filter irrelevant columns

3️⃣ **Exploratory Data Analysis**

* Statistical summaries
* Correlation analysis
* Data visualizations

4️⃣ **Visualization**
Charts and graphs were created using **Matplotlib** to highlight trends and patterns.
